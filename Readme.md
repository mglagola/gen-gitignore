# Easily generate .gitignore files

It's as easy as

```bash
$ genignore node sass
```

![demo](https://cl.ly/103c273R313Q/Screen%20Recording%202017-05-14%20at%2009.56%20AM.gif)

## Setup

```bash
$ npm install -g gen-gitignore
```

## Usage

```
Usage
  $ genignore <gitignore-names...>

Examples
  $ genignore swift ruby
  $ genignore android
  $ genignore node sass
```
